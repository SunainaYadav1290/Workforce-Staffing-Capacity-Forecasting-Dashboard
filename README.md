# Workforce-Staffing-Capacity-Forecasting-Dashboard
Built a complete WFM forecasting solution using historical call volumes and operational KPIs.
Applied moving average and seasonal index forecasting to estimate upcoming workload.
Designed capacity model using AHT, shrinkage, occupancy, and SLA requirements.
Built a Excel dashboard showing required staffing, dependability KPIs, and staffing gaps.
Delivered insights that help reduce under-staffing risks and optimize workforce utilization.
This project  includes:
Call Volume Forecast sheet.
AHT forecast sheet.
Capacity Planning Forecast 2024 sheet.
Workload Forecast sheet.
Dashboard Sheet
